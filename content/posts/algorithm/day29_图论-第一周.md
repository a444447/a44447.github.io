## 图论搜索

图论的搜索最常见的就是dfs与bfs。

dfs就是 **朝着一个方向一直搜索，直到无法再搜索才回溯**，它的代码结构就像这样:

```c++
void dfs(参数) {
    处理节点,
    dfs(图, 选择的节点) //这里是递归
    回溯,然后撤销之间的操作.
}
```

bfs就是 **把节点按层进行搜索，当前层搜索完毕后再搜索下一层**。

```c++
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }
    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
```
